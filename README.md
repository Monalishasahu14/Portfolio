# Personal Portfolio

This is a **personal portfolio website** built using **HTML, CSS, and JavaScript**. It showcases my projects, skills, and contact information in a clean, modern, and responsive design. The portfolio is fully mobile-friendly and interactive, designed to leave a professional impression.

---

## Features

- **About Me Section** – A brief introduction and personal bio.  
- **Projects Section** – Showcase of projects with images, descriptions, and links.  
- **Skills Section** – Display of technical skills with visual indicators.  
- **Contact Form** – Users can send messages via a contact form.  
- **Responsive Design** – Works seamlessly on mobile, tablet, and desktop.  
- **Interactive Elements** – Smooth animations, hover effects, and transitions.

- ## Tech Used

- **HTML** – Structure of the website  
- **CSS** – Styling and responsive layout  
- **JavaScript** – Interactivity and animations  
- **Git & GitHub** – Version control and repository hosting
